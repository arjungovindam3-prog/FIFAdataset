# ⚽ FIFA Dataset Analysis

This project focuses on exploring and analyzing a FIFA dataset using Python. It includes data cleaning, visualization, and insights extraction to better understand player attributes and trends.

---

## 📂 Project Structure

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Player performance insights  
- Visualization using charts and graphs
